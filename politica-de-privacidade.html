Introdução

Bem-vindo ao FinanceUp! A privacidade dos seus dados é nossa prioridade. Esta política explica como coletamos, usamos, armazenamos e protegemos suas informações pessoais quando você utiliza o aplicativo. Ao usar o FinanceUp, você concorda com os termos desta política.

1. Informações Coletadas

1.1 Informações Inseridas Manualmente

O FinanceUp não coleta automaticamente informações pessoais dos usuários. Os dados inseridos no aplicativo, como receitas e despesas, são exclusivamente fornecidos pelo usuário e armazenados localmente no dispositivo.

1.2 Informações Técnicas (Opcional)

Se o aplicativo for configurado para realizar backups ou sincronizações em nuvem (caso implementado no futuro), poderá ser necessário coletar:

  • Identificadores do dispositivo.
  • Credenciais para serviços de nuvem autorizados pelo usuário (ex.: Google Drive, Dropbox).

2. Como Utilizamos Suas Informações

Os dados fornecidos são utilizados exclusivamente para:

  • Ajudar você a gerenciar suas finanças de forma eficiente.
  • Gerar relatórios e análises financeiras dentro do aplicativo.

Nota: Não utilizamos seus dados para fins publicitários ou compartilhamos suas informações com terceiros.

3. Armazenamento e Segurança dos Dados

Os dados são armazenados localmente no dispositivo do usuário. O aplicativo não realiza upload automático de informações para servidores externos. Utilizamos as melhores práticas de segurança para proteger suas informações contra acesso não autorizado.

4. Compartilhamento de Dados

O FinanceUp não compartilha dados pessoais ou financeiros com terceiros, salvo se:

  • Exigido por lei ou ordem judicial.
  • Explicitamente autorizado pelo usuário (ex.: backups ou integrações futuras).

5. Direitos do Usuário

Você tem o direito de:

  • Excluir manualmente os dados armazenados no aplicativo.
  • Revogar permissões concedidas (se aplicável).
  • Solicitar suporte em caso de dúvidas sobre o funcionamento do app.

6. Alterações na Política de Privacidade

Podemos atualizar esta política de tempos em tempos para refletir melhorias no aplicativo ou alterações legais. Notificaremos os usuários sobre mudanças significativas por meio de atualizações no app.
